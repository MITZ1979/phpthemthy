HTML/CSS Structure

This theme is a responsive layout without sidebar. All of the information within the main content area is nested within a div with an class of "pm_wrapper". The general template structure is the same throughout the template. If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

  1. Open css/main.css.
  2. Find needed css-block.
  3. Change Value.

CSS Structure

We are using 4 CSS files in this theme. The first one is a main.css and grid.css files. This files also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

JavaScript

This theme next Javascript files:

  1. isotope.pkgd.min.js
  2. jquery.countdown.min.js
  3. jquery.jscrollpane.min.js
  4. jquery.magnific-popup.min.js
  5. jquery.min.js
  6. jquery.mousewheel.js
  7. kenburns.js
  8. main.js
  9. reflection.js
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Most of the animation in this site is carried out from the customs scripts.
  3. In addition to the custom scripts, I implement a few plugins to create the effects or get more functionality. This plugins is packed, so you won't need to manually edit anything in the file.
